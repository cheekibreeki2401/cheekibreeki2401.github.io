The Universal Tagging System

The project that got Edge on the map, the Universal Tagging System (UTS). The UTS was a piece of software made for Kyle Everett's final university project and over time has accumulated many interested people to fix the issue of files being lost or in different folders even though there is a commonality. The UTS has solved remote server issues and reports going missing thanks to the simple system that can be ran through the command line with CUTS and through a graphical user interface with GUTS.
